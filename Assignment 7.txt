Question 1: Write a query that counts all orders for October 3.

Query 1: SELECT COUNT(*) FROM orders 
	  WHERE order_date = '2026-10-03';

Query 2: SELECT COUNT(*) FROM orders 
	  WHERE MONTH(order_date) = 10 AND DAY(order_date) = 3;

Question 2: Write a query that counts the number of different non-NULL city values in the
Customers table.

Query: SELECT COUNT(DISTINCT city) AS unique_city_count
        FROM Customers;
--------------------------------------------------------------------------------------------------------
Question 3: Write a query that selects each customer’s smallest order

Query: SELECT cnum, MIN(amt) AS smallest_order
 	FROM orders
	GROUP BY cnum;
---------------------------------------------------------------------------------------------------------
Question 4: Write a query that selects the first customer, in alphabetical order, whose name
begins with G.

Query 1: Using ORDER BY and LIMIT
	 SELECT * FROM customers 
	 WHERE cname LIKE 'G%' 
	 ORDER BY cname ASC 
	 LIMIT 1;

Query 2: : Using a Subquery with MIN()
	 SELECT * FROM customers 
	  WHERE cname = (
   	 SELECT MIN(cname) 
   	  FROM customers 
   	  WHERE cname LIKE 'G%'
	  );

Question 5: Write a query that selects the highest rating in each city.

Query 1: SELECT city, MAX(rating) AS highest_rating
	 FROM Customers
	 GROUP BY city;

Query 2: SELECT * FROM Customers c1
         WHERE rating = (
         SELECT MAX(rating) 
         FROM Customers c2 
         WHERE c1.city = c2.city
         );

Question 6: Write a query that counts the number of salespeople registering orders for each
day. (If a salesperson has more than one order on a given day, he or she should be
counted only once.).

Query: SELECT order_date, COUNT(DISTINCT snum) AS active_salespeople
       FROM orders
       GROUP BY order_date;
