----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Question 1: Write a query that produces all pairs of salespeople who are living in the same city. Exclude combinations of salespeople with themselves as well as duplicate
            rows with the order reversed.

Query: SELECT s1.sname AS Salesperson_1, s2.sname AS Salesperson_2, s1.city
	FROM salespeople s1
	JOIN salespeople s2 ON s1.city = s2.city 
	WHERE s1.snum < s2.snum;

Question 2: Write a query that produces the names and cities of all customers with the same
rating as Hoffman.

Query 1: The Standard Subquery
	 
	 SELECT cname, city 
	   FROM Customers 
	   WHERE rating = (
   	 SELECT rating 
    	   FROM Customers 
    	   WHERE cname = 'Hoffman'
	   );

Query 2: Excluding Hoffman
   	 
    	SELECT cname, city 
	 FROM Customers 
	 WHERE rating = (
       SELECT rating 
    	 FROM Customers 
    	 WHERE cname = 'Hoffman'
	 )	 
	 AND cname <> 'Hoffman';
