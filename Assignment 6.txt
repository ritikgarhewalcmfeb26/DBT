Question 1: Write two different queries that would produce all orders taken on October 3rd or
	    4th, 1990.

Query 1: SELECT * FROM orders 
	 WHERE order_date IN ('1990-10-03', '1990-10-04');

Query 2: SELECT * FROM orders 
	 WHERE order_date = '1990-10-03' OR order_date = '1990-10-04';

Query 3: SELECT * FROM orders 
	 WHERE order_date BETWEEN '1990-10-03' AND '1990-10-04';

Question 2: Write a query that selects all of the customers serviced by Peel or Motika.
	    (Hint: the snum field relates the two tables to one another).

Answer: Query 1: Using a JOIN

	SELECT c.* FROM customers c
        JOIN salespeople s ON c.snum = s.snum
        WHERE s.sname IN ('Peel', 'Motika');
 	
	Query 2: Using a Subquery
	
	SELECT * FROM customers 
	WHERE snum IN (
    	SELECT snum 
    	FROM salespeople 
   	WHERE sname IN ('Peel', 'Motika')
	);

Question 3: Write a query that will produce all the customers whose names begin with a letter from ‘A’ to ‘G’.

Answer: SELECT * FROM customers 
        WHERE LEFT(cname, 1) BETWEEN 'A' AND 'G';
-------------------------------------------------------------------------------------------------------------------------
Question 4: Write a query that selects all customers whose names begin with the letter ‘C'.

Answer: SELECT * FROM customers 
	WHERE cname LIKE 'C%';
--------------------------------------------------------------------------------------------------------------------------
Question 5: Write a query that selects all orders except those with zeroes or NULLs in the amt field.

Query 1: SELECT * FROM orders 
	 WHERE amt <> 0 AND amt IS NOT NULL;
	
Query 2: SELECT * FROM orders 
	 WHERE IFNULL(amt, 0) <> 0;
